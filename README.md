# CEYORA

This project was developed as a basic web development assignment for NSBM Green University. 
It is a beginner-level landing page created using HTML, CSS, and JavaScript to demonstrate fundamental front-end web development concepts.

## Live Demo

The project is deployed using GitHub Pages and can be accessed using the following link: 
Website: 
https://ijan99.github.io/CEYORA/

A QR code was also generated using the GitHub Pages URL, allowing users to quickly access the website by scanning it with a mobile device.

## Features

- Responsive landing page
- Navigation
- Product tabs
- WhatsApp ordering
- Why Choose section
- About section
- Customer reviews
- Contact section

## Technologies

- HTML5
- CSS3
- JavaScript

## Purpose

The main objective of this project is to practice and demonstrate the 
basic concepts of front-end web development, including page structure, styling, interactivity, and responsive design.
